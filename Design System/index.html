What is a design system?

Greetings! A design system is a collection of files, articles, guilines, code snippets and other assets that a company or programmer uses. In this case this is the design system I will be using for my next Codeacdemy projects.

Fonts

When making a website unique one of the most important things along color are fonts. Serif fonts are often used in large text because they are easier to read than the sans-serif ones, which are often used in headings and appealing parts of the website. You can use a handwriting font if you want something more elegant or a monospace one if you are writing code! Remember to use web safe fonts as an alternative to downloaded ones.
