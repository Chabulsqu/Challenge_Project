Welcome!

HTML (Hyper Text Markdown Language) and CSS (Cascade Style Sheeting) are both extremely used languages along the web, used on things like the page you're seeing right now, but how do they work? All websites are made of an html containing the text and media of the web, as well as the metadata for search engines and browsers like the one you are using. Html is made out of tags en closed in <>. Some of the most used tags in the internet are the <h1>-<h6> for headings, <img> for images and <p> for paragraphs like this. Css is used for the graphic design of the web sites and it's the reason why we have nice looks on every page.


HTML Forms

Html forms can be confusing at the beginning, if you don't know how to make one then check this table. Be sure to always include opening and closing < form> tags and a <button> one to send the information.

Form Syntax Function Example
Text <input type="text" name="example"> Shows a text field that can be completed using letters and numbers
Password <input type="password" name="example"> Shows a text field where user can enter a password without this being visible
Numbers <input type="text" name="example"> Shows a number-only field that can be filled using two arrows buttons
Range <input type="range" name="example" min="1" max="10" step"0,5"> Shows a bar that users can slide to represent a value
Radio Buttons <input type="radio" name="example" value="example"> Shows a button that can only be clicked once at a time
Checkboxes <input type="checkbox" name="example" value="example"> Shows multiple checkboxes that can be selected at the same time
Remember to always specify a name for the information that is going to be sent to the server with the "name" attribute

Time for Design!

Css properties and selectors can be kind of tricky, but with enough dedication you will be able to understand them all. You can add Css directly into a html document either using the attribute "style" or by using the <style> tag.

Property Values Function Example
Color/background-color rgb, hexadecimal, hsl colors or keywords Changes either the font or the background's color of the element
Font-family One or more fonts, either web safe or imported from a site like Google Fonts Changes the default font design of the element
Font-weight numerical value (0 to 1000) or keywords like bold or light Changes the weight of an element's text
Text-align left, center, right or justify Chanfes the way an element's text is alligned
Border X number of px, border style (solid, ect) and color Changes the thickness, style and color of the border of an element
Margin A px value like "100 px" or "200px 500px" to change both vertical and horizontal margin. Specifies the margin from the element
Remember to always use {} curly brackets at the start and end of a declaration
